# !AAARG

**Framework open source pour ARG civiques**

---

## C'est quoi ?

!AAARG est un framework low-tech (HTML/CSS/JS pur) pour créer des jeux en réalité alternée (ARG) à visée d'éducation citoyenne.

- **Open source** — Code sous AGPL-3.0, contenu sous CC-BY-SA
- **Low-tech** — Zéro backend, zéro dépendance, fonctionne offline
- **Forkable** — Clone et adapte pour ta communauté
