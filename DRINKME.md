## Résumé (à adapter pour le formulaire)

!AAARG est un framework open source pour créer des ARG (jeux en réalité alternée) civiques. 

Stack : HTML/CSS/JS pur — zéro backend, zéro dépendance. Fonctionne offline, sur vieux hardware. Forkable.

Le projet vise à développer :
1. Un moteur de scénarios branching
2. Un système d'économie interne fictive
3. Une documentation pour facilitateurs

Tout sera publié sous AGPL-3.0 (code) et CC-BY-SA (contenu).
